Welcome to my Interactive Terminal Curriculum Vitae! 🚀 

Instead of a traditional static PDF, I built a web-based command-line interface (CLI) where you can explore my professional experience, education, and skills by typing commands just like you would in a real terminal.

👉 **[Live Demo: Explore my CV here](https://alioli97.github.io/about-me/)**

---

## ✨ Features

- **⌨️ Authentic CLI Experience:** Auto-focus, command history, and smooth auto-scrolling to keep you immersed.
- **🎨 6 UI Themes:** Switch between different vibes using the `theme` command (Dark, Light, Matrix, Dracula, Ubuntu, Retro).
- **🌍 Multi-language Support:** Fully localized in English (`en`), Spanish (`es`), and Catalan (`ca`).
- **📱 Fully Responsive:** Works perfectly on mobile devices with word-wrap and adaptive input fields.
- **🐰 Easter Eggs:** Try finding the hidden commands that aren't listed in the help menu! 👀

## 🛠️ Available Commands

| Command | Description |
| :--- | :--- |
| `help` | Lists all available public commands |
| `about` | A brief introduction about who I am |
| `experience`| Displays my professional work history |
| `education` | Shows my academic background & certifications |
| `skills` | Lists my technical skills and languages |
| `contact` | Prints my email and professional links |
| `theme [name]`| Changes the terminal UI (e.g., `theme dracula`, `theme matrix`) |
| `lang [en\|es\|ca]`| Changes the terminal language |
| `clear` | Clears the terminal screen |

*Hint: There are some hidden Linux-like commands. Try your luck!* 🕵️‍♂️

## 🚀 Tech Stack

This project was built from scratch using:
- **Vue 3** (Composition API) for reactive state management (command history, i18n, etc.)
- **Vite** for lightning-fast bundling and development.
- **Pure CSS3** utilizing CSS Variables for seamless theme switching.
- **GitHub Pages** & `gh-pages` package for automated CI/CD deployment.
